# Tingley
Show off and manage your LAN-Party. Let Tingley do the magic for you.

# How to Contribute

- Clone this Repository into a LAMP-Stack reachable location.
- Make sure there is a free database within your LAMP-Stack accessable.
- Follow the installation instructions.
- Code somethings great :)

More Docs are tracked down, just a few more steps...

# Origin

Tingley originates from the open source platform LAN Party Manager 4.2. Great thanks to the developers for their effort until now.
